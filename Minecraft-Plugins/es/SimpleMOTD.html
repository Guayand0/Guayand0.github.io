#   update-message: activar o desactivar el mensaje de actualizacion disponible al entrar al servidor
#   no-argument-message: activar o desactivar el mensaje del comando "/sm" en el caso de no tener ningun argumento
#   no-perm-message: activar o desactivar el mensaje para cuando usas un comando y no tienes permiso
#   show-icon: activar o desactivar icono
#   show-motd: activar o desactivar motd
#   maintenance-mode: activar o desactivar el modo mantenimiento
#   maintenance-mode-kick: echa a un jugador que intente entrar cuando esta el modo mantenimiento activado
