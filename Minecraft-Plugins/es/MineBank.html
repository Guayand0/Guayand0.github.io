Características

Permite almacenar dinero en el banco.
Permite ganar dinero pasivamente teniendo dinero en el banco.
Permite perder dinero con intereses al sacar el dinero del banco.
Crea bancos con niveles y almacenamiento ilimitados.
Cada banco tiene su propio permiso.
Permite usar todas las variables en los mensajes.
Permite usar variables en la GUI del inventario.
Soporte para PlaceholderAPI en mensajes y GUI del inventario.

GUI del inventario personalizable (1.13+):
· Cambia el nombre del inventario
· Cambia el tamaño del inventario
· Cambia la posición del ítem
· Cambia el ítem predeterminado
· Cambia el nombre del ítem
· Cambia el lore del ítem
· Cambia la cantidad del ítem
· Cambia la textura de los cabezas de jugador
· Haz clic en el ítem para usar un comando
· Añade un permiso necesario para usar el comando
· Soporte para PlaceholderAPI.

Variables

Variables del plugin MineBank
Funcionan en mensajes e inventario
· %plugin% - Muestra el prefijo del plugin
· %version% - Muestra la version actual del plugin
· %latestversion% - Muestra la ultima version del plugin
· %link% - Muestra el link de descarga del plugin
· %author% - Muestra el autor del plugin

Funcionan en mensajes e inventario
· %playerName% - Muestra el nombre del jugador que usa el comando.
· %playerBankBalance% - Muestra dinero actual en el banco del jugador que usa el comando.
· %playerBankLevel% - Muestra el nivel del banco del jugador que usa el comando.
· %playerBankName% - Muestra el nombre del banco del jugador que usa el comando.
· %playerBankMaxStorage% - Muestra el maximo de dinero del banco del jugador que usa el comando.
· %playerBankMaxLevel% - Muestra el maximo de nivel del banco del jugador que usa el comando.
· %playerBankTop% - Muestra pa posicion del top del banco del jugador que usa el comando.
· %playerBalance% - Muestra la cantidad de dinero de economia del jugador que usa el comando.

· %unlockNextLevelPrice% - Muestra la cantidad de dinero para subir de nivel el banco del jugador que usa el comando.
· %unlockNextLevel% - Muestra el nuevo nivel de banco al subir de nivel del jugador que usa el comando.

Funcionan solo en mensajes
· %targetName% - Muestra el nombre del jugador que recibe el comando.
· %targetBankBalance% - Muestra el dinero actual en el banco del jugador que recibe el comando.
· %targetBankLevel% - Muestra el nivel del banco del jugador que recibe el comando.
· %targetBankName% - Muestra el nombre del banco del jugador que recibe el comando.
· %targetBankMaxStorage% - Muestra el maximo de dinero del banco del jugador que recibe el comando.
· %targetBankMaxLevel% - Muestra el maximo de nivel del banco del jugador que recibe el comando.
· %targetBalance% - Muestra la cantidad de dinero de economia del jugador que recibe el comando.

Funcionan solo en mensajes
· %topPosition% - Posicion del jugador en el top del banco.
· %topName% - Nombre del jugador en el top del banco.
· %topBank% - Nombre del banco del jugador en el top del banco.
· %topBalance% - Dinero del jugador en el top del banco.

Funcionan solo en inventario
· %bankTopPosition<nº>% - Posicion del jugador en el top <nº> del banco.
· %bankTopName<nº>% - Nombre del jugador en el top <nº> del banco.
· %bankTopBalance<nº>% - Nombre del banco del jugador en el top <nº> del banco.
· %bankTopBankName<nº>% - Dinero del jugador en el top <nº> del banco.

Funcionan en mensajes y algunos tambien en inventario
· %amount% - Cantidad especificada en el comando.
· %setBankLevel% - Nivel establecido en el comando.
· %amountDeducted% - Cantidad retirada del banco sin intereses.
· %amountReceived% - Cantidad recibida al retirar dinero del banco con intereses.
· %interestPercentage% - Porcentage de intereses al retirar dinero del banco.
· %keepInBankProfit% - Cantidad recibida por tener dinero en el banco.
· %profitPercentage% - Porcentage de beneficio por tener dinero en el banco.
· %minAmountToWinProfit% - Minima cantidad de dinero en el banco para ganar beneficio.

Comandos

Los comandos azules son para administradores.

Los comandos verdes son para todos los jugadores.


Alias - minebank, mb

/minebank help - Muestra la lista de comandos de MineBank

/minebank reload - Recarga la configuración del plugin

/minebank version - Muestra la versión del plugin

/minebank author - Muestra el autor del plugin

/minebank permissions - Muestra la lista de permisos del plugin

/minebank plugin - Muestra el enlace de descarga del plugin


/bank - Abre el inventario (1.13+)

/bank help - Muestra la lista de comandos de Bank

/bank add <amount> - Deposita una cantidad de dinero

/bank add mid - Deposita la mitad del saldo actual de tu nivel de banco

/bank add midmax - Deposita la mitad del saldo máximo de tu nivel de banco

/bank add max - Deposita el saldo máximo de tu nivel de banco

/bank take <amount> - Retira una cantidad de dinero

/bank take mid - Retira la mitad del saldo actual de tu nivel de banco

/bank take midmax - Retira la mitad del saldo máximo de tu nivel de banco

/bank take max - Retira el saldo máximo de tu nivel de banco

/bank set <player> bal <amount> - Establece el saldo del jugador

/bank set <player> bal midmax - Establece el saldo del jugador a la mitad del saldo máximo de su nivel de banco

/bank set <player> bal max - Establece el saldo del jugador al máximo saldo de su nivel de banco

/bank set <player> level <level> - Establece el nivel del jugador

/bank set <player> level midmax - Establece el nivel del jugador a la mitad del nivel máximo del banco

/bank set <player> level max - Establece el nivel del jugador al nivel máximo del banco

/bank set <player> level <level> true - Establece el nivel del jugador y ajusta el saldo al máximo

/bank set <player> level <level> false - Establece el nivel del jugador y ajusta el saldo si está por encima del máximo

/bank top - Muestra los jugadores con más dinero en el banco

/bank bal - Consulta la cantidad de dinero de tu banco

/bank bal <player> - Consulta la cantidad de dinero del banco de otro jugador

/bank max bal - Consulta la capacidad máxima de dinero de tu nivel de banco

/bank max bal <player> - Consulta la capacidad máxima de dinero del banco de otro jugador

/bank max level - Consulta el nivel máximo de tu banco

/bank max level <player> - Consulta el nivel máximo de banco de otro jugador

/bank level - Consulta tu nivel de banco

/bank level <player> - Consulta el nivel de banco de otro jugador

/bank levelup - Mejora el nivel de tu banco

Permisos

Los permisos azules son para administradores.

Los permisos verdes son para todos los jugadores.


minebank.admin - Permiso de administrador

minebank.updatechecker - Permite recibir mensajes si hay una actualización

minebank.use - Permite usar el banco

minebank.bank.<bankName> - Establece el banco <bankName> para el jugador

Errores conocidos

BankInventoryEvent.openBankInventory: poner un ítem en un slot imposible (si hay 4 filas (9*4= 36) poner un ítem en el slot 42)

BankInventoryManager.reloadConfig: poner un idioma que no existe en "config.inventory-language" (por ejemplo, "qwe")
