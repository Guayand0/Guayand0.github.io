Características

Añade la durabilidad al lore de los items que son armas o armaduras.

Soporte para PlaceholderAPI.

Variables

Variables del plugin ToolTracker
Funcionan en mensajes y lore de items
· %plugin% - Muestra el prefijo del plugin
· %version% - Muestra la version actual del plugin
· %latestversion% - Muestra la ultima version del plugin
· %link% - Muestra el link de descarga del plugin
· %author% - Muestra el autor del plugin

Funcionan en mensajes y lore de items
· %currentDurability% - Muestra la durabilidad actual del item.
· %maxDurability% - Muestra la durabilidad maxima del item.

Variables de PlaceholderAPI
Funcionan en mensajes y lore de items

Comandos

Alias - tooltracker, tt

/tooltracker help - Muestra la lista de comandos de MineBank

/tooltracker reload - Recarga la configuración del plugin

/tooltracker version - Muestra la versión del plugin

/tooltracker author - Muestra el autor del plugin

/tooltracker permissions - Muestra la lista de permisos del plugin

/tooltracker plugin - Muestra el enlace de descarga del plugin

Permisos

tooltracker.admin - Permiso de administrador

tooltracker.updatechecker - Permite recibir mensajes si hay una actualización
