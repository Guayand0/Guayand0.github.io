Features

Adds durability to the lore of items that are weapons or armor.

Support for PlaceholderAPI.

Placeholders

ToolTracker plugin placeholders
Work in messages
· %plugin% - Shows the plugin's prefix
· %version% - Shows the current plugin version
· %latestversion% - Shows the latest plugin version
· %link% - Shows the plugin download link
· %author% - Shows the plugin's author

Work in item lore
· %currentDurability% - Shows the current item durability.
· %maxDurability% - Shows the maximum item durability.

Variables from PlaceholderAPI
Work in messages and item lore

Commands

Alias - tooltracker, tt

/tooltracker help - Shows the list of MineBank commands

/tooltracker reload - Reloads the plugin configuration

/tooltracker version - Shows the plugin version

/tooltracker author - Shows the plugin author

/tooltracker permissions - Shows the list of plugin permissions

/tooltracker plugin - Shows the plugin download link

Permissions

tooltracker.admin - Admin permission

tooltracker.updatechecker - Allows receiving messages when there is an update
