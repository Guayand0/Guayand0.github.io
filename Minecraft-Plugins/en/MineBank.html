Features

Allows storing money in the bank.
Allows earning money passively by keeping money in the bank.
Allows losing money with interest when withdrawing from the bank.
Creates banks with unlimited levels and storage.
Each bank has its own permission.
Allows using all placeholders in messages.
Allows using placeholders in the inventory GUI.
Supports PlaceholderAPI in messages and inventory GUI.

Customizable inventory GUI (1.13+):
· Change the inventory name
· Change the inventory size
· Change item position
· Change default item
· Change item name
· Change item lore
· Change item quantity
· Change player head texture
· Click on the item to execute a command
· Add a required permission to use the command
· Support for PlaceholderAPI.

Placeholders

MineBank Plugin Placeholders
Work in messages and inventory
· %plugin% - Displays the plugin prefix
· %version% - Displays the current plugin version
· %latestversion% - Displays the latest plugin version
· %link% - Displays the plugin download link
· %author% - Displays the plugin author

Commands

/minebank help

Shows the list of all MineBank commands.

/minebank reload

Reloads the MineBank configuration files.

/minebank setlevel [player]

Sets the bank level for a player.

/minebank setstorage [player]

Sets the bank storage amount for a player.

/minebank balance [player]

Shows the bank balance of a player.

/minebank deposit [player]

Deposits money into a player's bank.

/minebank withdraw [player]

Withdraws money from a player's bank.

/minebank pay

Transfers money from your bank to another player's bank.

Permissions

minebank.admin

Grants access to all administrative commands.

minebank.use

Allows use of bank commands.

minebank.create

Allows creation of new banks.

minebank.access.

Allows access to a specific bank.
